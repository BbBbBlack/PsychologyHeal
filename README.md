# PsychologyHeal
* 在服务外包开发比赛中开发的心理相关的App
* 主要功能有聊天，阅读文章，创建订单以及不同用户对订单的各种处理，心理测试，查看档案等
* 用户注册登录后可以查看文章、关注作者、收藏文章，选择咨询师并创建预约订单进行预约、与咨询师聊天等
* 咨询师登录后可以更改订单信息和状态，与用户交流等。聊天开发主要基于环信通信SDK，档案采用hellocarts
